## 说明

该项目为[《React 边学边做》]()课程项目，仅用于学习，禁止商业用途.

## 使用

1. 下载到本地

2. 切换到项目根目录

3. 项目初始化

```shell
npm install
# or
yarn install
```

4. 运行

```shell
npm run dev
# or
yarn dev
```

> ps: 项目请求数据需要 [react-store-api](https://github.com/lirenmi/react-store-api) 项目提供
